Grizzly-Blink
=============

[Grizzly NIO framework](http://grizzly.java.net/) filters to process the
 [Blink](http://blinkprotocol.org/) protocol serialized messages.

Grizzly-Blink is created and maintained by Chris Molozian (@novabyte).
